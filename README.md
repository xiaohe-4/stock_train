# THU-BDC2026-main
基于Transformer的排序学习选股框架

> 2026中国高校计算机大赛——大数据挑战赛

本项目为2026年大数据挑战赛提供了一套完整的排序学习选股解决方案。核心任务是基于沪深300指数成分股的历史量价数据，构建深度学习模型预测未来5个交易日收益率最高的Top-5股票组合。


## 📌 项目概述

比赛要求：基于沪深300指数成分股的历史股价数据，通过建立机器学习模型来预测未来一周收益最大的股票组合（不超过5只，累计权重不超过1），以此进行排名。

本框架实现了：
- **时序建模**：Transformer Encoder提取60个交易日序列特征
- **股票间交互**：自定义Cross-Stock Attention模块捕捉股票横向对比关系
- **排序学习**：Listwise + Pairwise组合损失函数，着重优化Top-5选股精度

# 模型是否能学习每周涨幅前五存在的隐形特征？

## 📈 模型能学习什么？——从“显性”到“隐性”的挖掘
模型无法直接“看到”一个神秘的“大涨标签”，而是通过大量数据，尝试构建一个从各种输入特征到未来收益的复杂映射关系。这个过程中，它确实可能发现一些人类不易察觉的模式。

捕捉非线性关系：传统线性模型难以处理特征间的复杂交互，而深度学习模型（如DNN、LSTM等）擅长捕捉这些非线性关系。例如，模型可能发现“某类技术指标在特定宏观环境下”的组合才有效。

挖掘高维数据中的模式：模型可以处理海量数据，如高频量价数据。通过Logsig-RNN等模型，能将高频数据流转化为特征；通过TCN、TimesNet等卷积模型，可以捕捉时间序列中的短期趋势和局部模式。

学习股票间的“隐性关系”：通过图神经网络（GNN）等模型，可以学习股票之间动态的、非显性的关联。如FinGAT模型能自动学习股票与板块间的潜在交互，其他研究也表明GNN能挖掘公司间的隐性关系。

构建“隐性因子”：深度学习可以直接从数据中构建“隐性因子模型”。研究表明，这类模型在A股市场的预测表现显著超越了传统基准模型。

## ⚠️ 为何“隐形特征”难以捉摸？——面临的挑战
尽管模型强大，但预测“每周涨幅前五”面临几个根本性挑战。

信噪比极低：金融数据中噪声远大于有效信号。短期（周度）价格波动受大量随机因素影响，模型极易学到无经济意义的噪声。

市场非平稳性：股票市场的统计规律会随时间变化。今天有效的模式，明天可能就失效，这使得模型容易过时。

过拟合风险高：在低信噪比下，模型很容易“记住”历史中的偶然模式，导致样本外表现不佳。

因果推断的困难：模型找到的相关性不等于因果关系。即使模型找到某个特征与涨幅相关，也可能只是虚假相关，缺乏可靠的预测能力。

数据与特征瓶颈：模型的性能高度依赖于输入数据的质量和特征工程。如果缺乏关键信息（如内幕消息），模型的上限会被锁定。
